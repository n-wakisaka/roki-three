roki-three pages viewer
使い方
  • Pair は「ztk + zvs」を1組として読み込む単位です。
  • Add pair を押すと、新しい Pair を追加できます。
  • 各 Pair のチェックボックスは「Load selected の対象に含めるか」を表します。
  • Pair 0 は同梱サンプル（example/arm）です。不要ならチェックを外してください。
  • 読み込みは、必要な Pair をチェックして Load selected を押します。
  • Clear all は、読み込み済みモデルをシーンからすべて削除します（Pair入力欄は残ります）。
  • 再生は Play/Pause/Restart で操作します。
  • Save / Load config で pair 構成を保存・復元できます。file 指定の場合は内容は保存されません。
